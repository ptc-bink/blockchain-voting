# BlockChainVoting

A blockchain-based E-voting system, developed for the Final year project at Bhagubhai Mafatlal Polytechnic, Mumbai. This project was made with the help of Sayyam Gada and Charmee Mehta.

Technologies used:
- Solidity
- Next.js
- NodeMailer
- Semantic UI React
- MongoDB
- ExpressJS
- Web3
- Metamask Extension
- AJAX
